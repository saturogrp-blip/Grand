### Admin Panel

|--------|-------|--------|
| **Add Curator** | Type name ‚Üí Click "+ Add" | Curator added to org |
| **Remove Curator** | Click "‚úï Remove" ‚Üí Confirm | Curator deleted |
| **Change Senior** | Type name ‚Üí Click "Update" | Global senior updated |
| **Export Data** | Click "Export Curators" | JSON downloads |
| **Import Data** | Click "Import" ‚Üí Select file | Curators restored |
| **Reset All** | Click "Reset Curators" | Back to defaults |

### JavaScript API

```javascript
// Get curators for organization
CuratorDB.getCurators('FIB')

// Add a curator
CuratorDB.addCurator('LSPD', 'John Doe')

// Remove a curator
CuratorDB.removeCurator('SAHP', 'Lilith')

// Update senior curator
CuratorDB.setSeniorCurator('New Name')

// Get senior curator
CuratorDB.getSeniorCurator()

// Export as JSON
const json = CuratorDB.exportToJSON()

// Import from JSON
CuratorDB.importFromJSON(jsonString)

// Get all data
CuratorDB.getAllCurators()

// Reset to defaults
CuratorDB.resetToDefaults()

// Get statistics
CuratorDB.getStats()
```

## üéØ Common Workflows

### Workflow: Update FIB Team
```
1. Open admin.html
2. Select "FIB" from dropdown
3. Click remove (‚úï) for each old curator
4. Add new curators one by one
5. Verify in question generator
```

### Workflow: Backup Data
```
1. Open admin.html
2. Click "Export Curators"
3. Save file with date: curators_2025-12-03.json
4. Store safely (email, cloud, version control)
```

### Workflow: Restore from Backup
```
1. Open admin.html
2. Click "Import Curators"
3. Select backup JSON file
4. Data restored automatically
```

### Workflow: Sync Multiple Browsers
```
Device A: Export curators
Transfer: Email/USB/Cloud
Device B: Import curators
Both devices now have same data!
```
